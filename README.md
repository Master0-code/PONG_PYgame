# PONG_PYgame
Game created using python and pygame. sounds are generated using math and array libs(beeps only). 

# Pong

A local two-player Pong game made with Python and Pygame.

## Features

- Local 2-player gameplay
- Sound effects generated in Python
- Pause system
- Scoreboard
- Window resizing

## Controls

Player 1
- W = Up
- S = Down

Player 2
- ↑ = Up
- ↓ = Down

Mouse
- Click = Pause / Resume

## Installation

```bash
pip install pygame-ce
python main.py
```
